This tool is used to send comands to the voyager spacecraft, and to receive data from it.

this tool receive one field:

- command: the command to send to the voyager spacecraft

commands can be:

- "turn_on": to turn on instrumments of the voyager spacecraft
- "turn_off": to turn off instrumments of the voyager spacecraft
- "get_status": to get the status of the voyager spacecraft
